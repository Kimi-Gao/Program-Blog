## transform-decorators

在Babel v6 中并不支持Decorators特性, 更多 – babel/babel#2645.
### Installation

``` sh
$ npm install babel-plugin-transform-decorators
```
### Usage

在`.babelrc`中添加以下配置:

``` json
{
  "plugins": ["transform-decorators"]
}
```
